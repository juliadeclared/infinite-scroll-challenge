## Getting Started

### Dependencies

- nodemon
- Material UI
- Framer Motion
- Express
- Axios

### Installing

- First install dependencies in the root directory

```
yarn

```

- Then cd into server and install dependencies again

```
cd client
yarn
```

### Executing program

- From the root directory, run

```
yarn dev
```
